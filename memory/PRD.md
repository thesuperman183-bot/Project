# Aditya Anand — Portfolio Website PRD

## Original Problem Statement
Build a modern, responsive personal portfolio for Aditya Anand — minimalist, futuristic, nature-inspired ("a future quantum researcher exploring technology from within a peaceful misty forest"). Light mode with misty forest + drifting fog; dark mode with moonlit forest + fireflies. Sections: Hero, About, Core Strengths, Learning Journey (progress bars), Certifications, Why Consider Me, Future Roadmap (timeline), Projects, Contact, Footer.

## User Persona
- **Aditya Anand** — B.Tech CSE (AI & ML) student at UpGrad School of Technology, future quantum computing researcher.

## Architecture
- Frontend-only static portfolio (React + Tailwind + framer-motion-style CSS animations + shadcn/sonner toasts).
- No backend storage needed (static contact buttons only).
- Components organized under `/app/frontend/src/components/portfolio/`.

## Implemented (Dec 2025)
- Animated misty forest background (light) + moonlit forest with 32 blinking fireflies (dark).
- Sticky glass nav with dark mode toggle.
- Hero with profile photo, name, intro, pill buttons (Resume → toast "under construction", Contact).
- About section with interest tags.
- 3 core strength cards (Curiosity, Self-Belief, Mindfulness).
- Learning Journey with animated progress bars (4 groups, 10 skills).
- Certifications: Python Quick Start, ADCA, "Coming Soon" card.
- Why Consider Me (5 glass cards).
- Future Roadmap vertical animated timeline (6 stages).
- Projects grid (Portfolio, AI, Quantum, Research).
- Contact section: LinkedIn / Email / WhatsApp pill buttons (real links).
- Footer with quote, social icons, back-to-top.
- SEO meta tags, smooth scroll, scroll-reveal observer, accessible data-testids.
